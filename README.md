# Data Visualization and Analytics Laboratory
 
This repository contains a complete set of laboratory experiments focused on data
visualization, exploratory data analysis, multivariate analysis, dashboard design,
and handling large-scale datasets using **R** and **Power BI**.
 
---
 
## Experiment 1: Implementation of Data Charts
 
### Objective
To design appropriate data charts to analyze and compare academic performance indicators.
 
### Scenario
An autonomous engineering college analyzes internal assessment performance of first-year students across multiple subjects to identify difficult courses and improve teaching strategies.
 
### Tasks Performed
- Import student performance dataset
- Perform basic data preprocessing
- Create bar chart for subject-wise average marks
- Generate line chart for Internal Test performance trends
- Plot pie chart for grade distribution
 
### Tools & Technologies
- R  
- ggplot2  
- dplyr  
- tidyr  
 
### Outcome
Clear visualization of student performance trends and identification of low-performing subjects.

## Experiment 7: Over-Plotting Reduction Techniques
 
### Objective
To apply techniques for reducing over-plotting in large datasets and improve visualization clarity.
 
### Scenario
A social media analytics company visualizes millions of user interactions to study engagement patterns.
 
### Tasks Performed
- Normal scatter plot analysis
- Alpha blending
- Jittering techniques
- Aggregation and binning methods
 
### Tools & Technologies
- R  
- ggplot2  
 
### Outcome
Clear visualization of dense data by reducing overlap and highlighting data density patterns.
